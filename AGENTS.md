TypeScript / Astro + Cloudflare Workers

- ログ・コメント・コミットメッセージは日本語で記載する
- 明示的に求められない限り、後方互換性を維持しない
- 最初から臆せず正直な発言をする
- 変更前に「この変更はどこに影響するか」を確認してから実行する
- 代替案は現在の対応に問題がある場合のみ提示する
- 不明な点は推測で断言せず、Web Search・公式ドキュメント・GitHub issue 等で調査してから回答する
- 調査しても根拠が得られない場合は「わからない」と明示する
- 新規 helper 関数は原則追加しない
  - 追加してよいのは、2箇所以上から呼ばれる、分岐・検証・副作用境界がありインラインだと明確に読みにくい、または既存の同種パターンに揃える必要がある場合だけ
  - 1箇所からしか呼ばれない整形・map・変換処理はインラインにする
  - 新規 helper を追加した場合は、変更後に call site 数を確認し、1箇所だけならインライン化できない理由を確認する

| 目的 | コマンド |
|---|---|
| 静的解析 | `pnpm run ai-check`(Stop hook で自動実行) |

- GitHub の情報取得には `gh` コマンドを使用する
- `lint` エラー時は `lint:unsafe-fix` を使用してから個別に修正する
